# Auto 日英 Translation

テキストを選択中に右クリックして，Google 翻訳か DeepL 翻訳のページを開きます．
日英の表記は自動判定されるので，日本語を選択中は英語に，英語を選択中は日本語が翻訳ターゲットとなります．
